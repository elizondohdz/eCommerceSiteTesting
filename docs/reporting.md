# HTML Reporting

## Overview

The reporting for this project is made using mochawesome reports, this tool allow the project to generate:

- Logs for the tests
- Screenshots
- A quick video from the failed tests

---

## Overview Report

Generated HTML report for the test run execution this report contains:

- Amount of passed and failed tests
- Amount of time each tests needed to finish
