# Backend 

## Overview

This project was built using Laravel 13 and it contains

- API
- Authentication
- Unit tests with PHPUnit
- Integration tests

---

## Features

If you are looking for a specific feature, the list is:

Auth
- User Authentication with JWT (JWT was used cause it is the standard, but I could also be using Sanctum)

DB
- Database created with migrations
- Database Seeders
- Model Factories
- Entity Relationship Model DB

Controllers
- SOLID Principles
- Use of specific requests for validations
- Use of specific resources to secure the response objects

Unit Tests
- In memory Database
